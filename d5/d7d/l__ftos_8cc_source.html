00022  * float to string
00023  * builds string representing floating number
00024  * num = number to convert.
00025  * str = string to put it in.  Must be big enough, or else!!
00026  *          Must have length at least len+6.
00027  *          sig digits + dec.pt.(1) + sign(1) + exp(4)
00028  * len = max number of displayed digits. (left + right of dp)
00029  *          (includes d.p. if 'e' notation and 2 digit exp)
00030  * fmt = format : 0       = alpha for exp notation
00031  *                FMTEXP  = exponential notation
00032  *                FMTSIGN = always inlcude sign
00033  *                FMTFILL = fill in zeros
00034  * //BUG//
00035  * returns a pointer to static space where the string is.
00036  * there is a finite pool, so repeated calls work, to a point.
00037  * after that, the space is overwritten, every POOLSIZE calls
00038  */
