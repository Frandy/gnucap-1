00227 // Respectively you can give a parameter a statistical dispersion function,
00228 // either rectangular distribution 
00229 //      UNIF(nom_val, rel_var)
00230 //      AUNIF(nom_val, abs_var)
00231 // or normal distribution: 
00232 //      GAUSS(nom_val, rel_var, sigma)
00233 //      AGAUSS(nom_val, abs_var, sigma)
00234 // Normal distribution can be represented with variation values of 1, 2 or 3 sigmas.
00235 // In both cases rel_var=abs_var/nom_val.
00236 // for now, stubs, return nom_val
