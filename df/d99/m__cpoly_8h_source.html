00022  * structs for fixed order polynomials, in 2 different forms
00023  * FPOLY is by function -- f0 = eval of function
00024  *                         f1 = 1st derivative
00025  *                         f2 = 2nd derivative
00026  *                         etc.
00027  *                      f(x) = f0
00028  *                      f(t) = f0 + f1*(t-x) + f2*(t-x)^2 + ...
00029  * CPOLY is by series -- c0 = coeff of x^0 term (constant)
00030  *                       c1 = coeff of x^1 term (1st derivative)
00031  *                       c2 = coeff of x^2 term
00032  *                       etc.
00033  *                      f(x) = c0 + f1*x + f2*x^2 + ...
00034  *                      f(t) = c0 + f1*t + f2*t^2 + ...
00035  */
